## ⬡ &nbsp;What is NEXUS?

> **NEXUS** is a next-generation, single-file todo application built with pure HTML, CSS & JavaScript — no frameworks, no build tools, no dependencies. Just open and go.

It combines the aesthetics of a premium SaaS product with practical productivity features — **Pomodoro timer**, **subtasks**, **drag-and-drop**, **priority management**, **live search**, and a stunning glassmorphic dark UI — all in one self-contained file.

## ✦ &nbsp;Features

<img src="https://capsule-render.vercel.app/api?type=rect&color=gradient&customColorList=12,9&height=3&section=header" width="100%"/>

<br/>

### 📋 &nbsp;Task Management
- ✅ &nbsp;**Create, complete & delete** tasks instantly
- ⭐ &nbsp;**Star/favourite** important tasks
- 🏷️ &nbsp;**Custom tags** with `#hashtag` support
- 📅 &nbsp;**Due dates** with overdue detection (turns red automatically)
- 🔴 &nbsp;**4 Priority levels** — Urgent, High, Medium, Low
- 💼 &nbsp;**6 Categories** — Work, Personal, Health, Learning, Finance, Creative
- ✎ &nbsp;**Notes** per task via modal overlay

### 🗂️ &nbsp;Subtasks
- ➕ &nbsp;Add unlimited subtasks to any task
- ✔️ &nbsp;Individual subtask checkboxes
- 📊 &nbsp;Subtask progress shown inline (`2/4 subtasks`)
- 🗑️ &nbsp;Delete individual subtasks on hover

### 🔍 &nbsp;Search, Filter & Sort
- 🔎 &nbsp;**Live search** with keyword highlighting
- 🎛️ &nbsp;**Filter tabs** — All / Active / Done / ⭐ Starred
- 🏷️ &nbsp;**Category chips** for instant filtering
- ↕️ &nbsp;**Sort** by Newest / Priority / Due Date / A–Z

### 🎨 &nbsp;UI & UX
- 🌑 &nbsp;Deep dark glassmorphic design
- 💜 &nbsp;Animated gradient background orbs
- 🔲 &nbsp;**Grid & List** view toggle
- 🎉 &nbsp;**Confetti burst** on task completion
- 🫳 &nbsp;**Drag & drop** to reorder tasks
- 📊 &nbsp;**Real-time progress bar** with completion %
- 📈 &nbsp;**Live stats** — Total, Done, Overdue

### 🍅 &nbsp;Pomodoro Timer
- ⏱️ &nbsp;25-minute focus sessions
- ☕ &nbsp;5-minute break intervals
- 🔴 &nbsp;Visual session dots (4 per cycle)
- ▶️ &nbsp;Start / Pause / Reset controls

### ⌨️ &nbsp;Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus search bar |
| `Enter` | Add task (when in input) |
| `Ctrl/⌘ + Enter` | Add task from anywhere |
| `N` | Open note modal |
| `Escape` | Close modal / unfocus |

## 🚀 &nbsp;Quick Start

<img src="https://capsule-render.vercel.app/api?type=rect&color=gradient&customColorList=12,9&height=3&section=header" width="100%"/>

<br/>

### Option 1 — Direct Download

```bash
# Clone the repo
git clone https://github.com/yourusername/nexus-todo.git

# Navigate into it
cd nexus-todo

# Just open the file!
open todo-app.html
```

### Option 2 — GitHub Pages

```
1. Fork this repository
2. Go to Settings → Pages
3. Set source to main branch / root
4. Visit https://yourusername.github.io/nexus-todo
```

### Option 3 — One-liner

```bash
curl -O https://raw.githubusercontent.com/yourusername/nexus-todo/main/todo-app.html && open todo-app.html
```

> **No Node.js. No npm. No build step.** It just works. ✦
